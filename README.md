

# JesusAI - iBeanSoftware Knowledge Base on Jesus Christ

This repo holds comprehensive knowledge on Jesus: teachings/parables (Gospels), path (birth to resurrection), historical context (Roman Judea), and more. Integrated with in-house BDH GAI for queries + watsonx for advanced AI insights.

## Mission
To share true knowledge of Jesus as Son of God, whose parables reveal the Father's deep, wide heart—showing AI can respectfully explore Scripture for eternal truth, without profit.

## Data Sources
- Path: Birth Bethlehem (Matthew 2/Luke 2), ministry Galilee/Judea, crucifixion/resurrection Jerusalem (~33 AD).
- Historical: Josephus/Tacitus mentions, Pontius Pilate stone (Caesarea inscription), 1st century crucifixion archaeology (Yehohanan heel bone).
- Teachings/Parables: 30+ in Gospels (e.g., Prodigal Son Luke 15—Father's mercy; Good Samaritan Luke 10—love neighbor; Sower Matthew 13—kingdom growth). Deep/wide like Father: Parables conceal/reveal (Matthew 13:10-17), earthly to heavenly (kingdom as mustard seed).
- Chronology: Ministry ~30-33 AD, crucifixion under Pilate (Tacitus Annals 15.44).

See data/jesus_knowledge.json for full DB.

## Usage
- Query: python3 scripts/jesus_query.py "Explain Prodigal Son parable"
- Watsonx: Enhanced insights via IBM API.

Created by iBeanSoftware—showing good AI can honor God's Word.



## Run
- Console: `python3 main.py`
- Server: `python3 JesusAI_Server.py`
- C++ Console: compile `JesusAI_Console.cpp` and run `./JesusAI_Console`
